# Privacy Policy — Timetable Creator

**Last updated:** June 2026

Timetable Creator is a Chrome extension that helps you build and manage a weekly timetable.

## Data Collection

Timetable Creator does **not** collect, transmit, or share any personal data.

All information you enter — class names, room numbers, schedule times, colors, and the timetable title — is stored using Chrome's built-in `storage.local` API. This data:

- Stays entirely on your own device, inside your own Chrome browser profile
- Is never sent to any external server
- Is never accessed by the developer of this extension
- Is never shared with or sold to any third party

## Permissions Used

- **storage** — used only to save your timetable locally so it persists between browser sessions.
- **tabs** — used only to open a new browser tab when you click "Print," so the print dialog can render correctly. No information about your other open tabs is read or collected.

## Third-Party Services

This extension does not use any analytics, advertising, or tracking services of any kind.

## Changes to This Policy

If this policy changes in the future, the updated version will be published along with the extension's listing on the Chrome Web Store.

## Contact

For questions about this privacy policy, please use the support contact listed on the extension's Chrome Web Store page.
